# Onboarding Guide — Catastro Project

This guide is specific to the Catastro project and is not managed by Brain.
It describes team-specific onboarding steps that are not part of the Brain core.

## Getting Started

1. Clone the repository
2. Run `npm install` to install dependencies
3. Copy `.env.example` to `.env` and fill in the required values
4. Ask a team member for the secrets needed to connect to the Catastro data service

## Project-Specific Setup

The Catastro team uses a custom database schema. See `brain/project/architecture/`
for the data model and `brain/project/decisions/` for architectural decisions.

## Contact

For questions about this onboarding guide, contact the Catastro team lead.
